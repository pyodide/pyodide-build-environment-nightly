Pyodide nightly cross-build environments

Nightly cross-build environments (xbuildenvs) for building Pyodide packages against unreleased tip-of-tree versions of Pyodide

Release builds

Standard cross-build environments, suitable for most package builds.

api/v2/release.json

Debug builds

Cross-build environments compiled with PYODIDE_DEBUG=1.

api/v2/debug.json

Install a nightly environment via pyodide-build: 

      
      pyodide xbuildenv install --nightly                  # latest release build
      pyodide xbuildenv install --debug                    # latest debug build
      pyodide xbuildenv install --nightly 20260520         # specific version
      pyodide xbuildenv search --nightly --debug --all     # search all versions
